# Boycott Siren - Workaround Version

## 🚨 What's Different?

This version uses **floating overlay alerts** instead of inline indicators.

### Why Workaround?

The main version tries to add indicators directly to page elements, but Ticketmaster's CSS sometimes hides them. This version solves that by creating **floating notifications** that appear in the top-right corner - impossible to miss!

## ✨ Features

- **Floating Overlay**: Unmissable notification box in top-right corner
- **Auto-dismiss**: Alerts disappear after 10 seconds
- **Israeli Flag**: Visual indicator with Star of David
- **Click for Details**: Full artist info modal
- **Maximum Z-index**: Appears above all page content

## 📦 Installation

1. Download and extract this folder
2. Open Chrome: `chrome://extensions/`
3. Enable "Developer mode" (top right)
4. Click "Load unpacked"
5. Select this folder
6. Enable "Automatically allow access on the following sites"

## 🎯 How It Works

When you visit **any supported ticket site** and a boycotting artist is detected:

1. **Floating alert slides in** from the right
2. Shows: Artist name, stance (RED/YELLOW), statement
3. **Click "View Sources & Details"** to open full info on website
4. **Click ×** to dismiss (or it stays until you close it)
